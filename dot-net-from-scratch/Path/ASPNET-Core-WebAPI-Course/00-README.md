## Suggested pace

| Stage | Topics | Suggested time |
|---|---|---|
| Beginner | Files 01–06 | 1–2 weeks |
| Intermediate | Files 01–08 | 2–3 weeks |
| Advanced | Files 01–10 | 3–4 weeks |

## Prerequisites

- Basic C# knowledge (variables, classes, loops, OOP basics)
- Basic understanding of HTTP (what a request/response is)
- A computer that can run .NET SDK (Windows, macOS, or Linux)

## What you'll be able to build by the end

A secured, tested, documented, containerized Web API with:
- EF Core + a real database
- JWT authentication and role-based authorization
- Global error handling and structured logging
- Caching, versioning, and Swagger docs
- Unit + integration tests
- A Dockerfile ready for deployment

## How to use this course

1. Read each file in order — later files assume earlier concepts.
2. Type out the code samples yourself instead of copy-pasting; it builds muscle memory.
3. After each stage, try the **"Practice Task"** at the bottom of the last file in that folder before moving on.
4. Keep one running project and evolve it stage by stage — don't start a new project per topic.
