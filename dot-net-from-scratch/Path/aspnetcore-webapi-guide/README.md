# ASP.NET Core Web API — Beginner to Advanced

A structured, self-paced learning path for building production-grade Web APIs with ASP.NET Core. Work through the folders in order — each one builds on the last.

## How to use this guide

- Read each `.md` file in order within a folder.
- Type out the code samples yourself instead of copy-pasting — it builds muscle memory.
- After each folder, try the "Practice Task" at the bottom of the last file in that section.
- Prerequisite: basic C# syntax (variables, classes, methods, LINQ basics). No prior web dev experience required.

## Suggested pace

| Stage | Folder | Time (approx.) |
|---|---|---|
| Beginner | `01-beginner` | 1 week |
| Intermediate | `02-intermediate` | 2 weeks |
| Advanced | `03-advanced` | 2–3 weeks |
| Expert | `04-expert` | 3–4 weeks |

## Tools you'll need

- **.NET SDK** (latest LTS) — https://dotnet.microsoft.com/download
- **Visual Studio 2022**, **VS Code**, or **JetBrains Rider**
- **Postman** or the built-in `.http` files for testing endpoints
- **SQL Server / SQLite / PostgreSQL** (from the Intermediate section onward)
- **Docker Desktop** (for the Expert section)

Good luck — start with `01-beginner/01-introduction.md`.
